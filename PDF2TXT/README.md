# PDF to TXT 
## 项目结构
PDF2TXT
    ├─pdf
    └─txt     
    ├── pdf2txt.py       # 你的转换脚本                          
    └── README.md              # 使用说明文档                        
## 关于使用            
This project converts EPUB files to TXT format.
#### Requirements

- Python 3.x
- requirement
```
pip install pdfplumber
```
#### run
需要进入到这个目录下
```
python pdf2txt.py 
```

## 总结
提供了一个简单、轻量级的解决方案，使用Python和第三方库pdfplumbe快速将PDF文件转换为TXT文件。
